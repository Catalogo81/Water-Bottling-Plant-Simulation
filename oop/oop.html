                class BankAccount {
                    // Private fields (Encapsulated)

                    // Public method to access balance (controlled)

                    }
                }

üîß Composition (HAS-A) - Building with Parts

Car HAS-A Engine üöó‚öôÔ∏è

A car is not an engine (IS-A), but a car HAS-A engine (composition)!

üè∑Ô∏è IS-A (Inheritance)

Relationship: Child IS-A Parent

Car IS-A Vehicle
Dog IS-A Animal
Circle IS-A Shape

üî© HAS-A (Composition)

Relationship: Object HAS-A Component

Car HAS-A Engine
House HAS-A Door
Phone HAS-A Battery 
                class Engine {
                    // Private fields (Encapsulated)
                    private String type;
                    private int horsepower;
                    
                    // Public method to start engine
                    void start() {
                        System.out.println("Engine started!");
                    }
                }

                class Car {
                    // Car HAS-A Engine (Composition)
                    private Engine engine;
                    private String brand;
                    
                    // Constructor initializes Engine
                    Car() {
                        engine = new Engine();
                    }
                    
                    // Public method to start car
                    void startCar() {
                        engine.start();
                        System.out.println("Car is ready to drive!");
                    }
                }
